# 占位符

## 通用

通用占位符：

| 占位符 | 描述                 |
| ------ | -------------------- |
| %v     | 显示默认值           |
| %#v    | 显示 Go 语法的默认值 |
| %T     | 显示 Go 语法的类型值 |
| %%     | 显示百分号           |
