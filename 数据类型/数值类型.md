# 整形

## 标准整形

Golang 中标准整形按长度进行分类：

|  类型  |                             描述                             |
| :----: | :----------------------------------------------------------: |
| uint8  |                 无符号 8 位整型 ( 0 到 255 )                 |
| uint16 |               无符号 16 位整型 ( 0 到 65535 )                |
| uint32 |             无符号 32 位整型 ( 0 到 4294967295 )             |
| uint64 |        无符号 64 位整型 ( 0 到 18446744073709551615 )        |
|  int8  |               有符号 8 位整型 ( -128 到 127 )                |
| int16  |             有符号 16 位整型 ( -32768 到 32767 )             |
| int32  |        有符号 32 位整型 ( -2147483648 到 2147483647 )        |
| int64  | 有符号 64 位整型 ( -9223372036854775808 到 9223372036854775807 ) |

其中有几个特别的标准整形：

- uint8：别名为 byte 类型，用来区分字符值和整数值
- int32：别名为 rune 类型，用来区分字符值和整数值
- int16：在 C 语言中，int16 就是 short 类型
- int64：在 C 语言中，int64 就是 long 类型



## 动态整形

动态整形是指在不同的平台上有着不同差异的整形：

|  类型   |                          描述                          |
| :-----: | :----------------------------------------------------: |
|  uint   | 32 位操作系统上就是 uint32，64 位操作系统上就是 uint64 |
|   int   |  32 位操作系统上就是 int32，64 位操作系统上就是 int64  |
| uintptr |              无符号整型，用于存放一个指针              |

注意！len 函数可以根据不同平台动态获取动态整形长度，但是在涉及到二进制传输、读写文件结构描述时，需要谨慎使用 uint 和 int。



## 进制写法

在 Go 1.13 版本后，Golang 支持根据字面量形式来书写不同的进制，如下所示：

```
package main

import "fmt"

func main() {
	var (
		bin int = 0b10 // 定义二进制数
		oct int = 077  // 定义八进制数
		hex int = 0xff // 定义十六进制数
	)
	fmt.Printf("%d \n", bin)
	fmt.Printf("%d \n", oct)
	fmt.Printf("%d \n", hex)
}

// 2
// 63
// 255
```

我们也可以借助 fmt.Printf 函数显示一个十进制数的其它进制表示：

```
package main

import "fmt"

func main() {
	var number int = 10          // 定义一个十进制数
	fmt.Printf("%d \n", number)  // 显示十进制
	fmt.Printf("%b \n", number)  // 显示二进制
	fmt.Printf("%o \n", number)  // 显示八进制
	fmt.Printf("%0x \n", number) // 显示十六进制
}

// 10
// 1010
// 12
// a
```





## 数字分割

Golang 可以使用 _ 对数字进行分割以便更加直观显示大数：

```
var number uint32 = 300_04

// 30004
```



# 浮点型

## float32

float32 最大能表示的值约为 3.4e38，可以使用 math.MaxFloat32 查看该值。

定义 float32 代码示例：

```
package main

import "fmt"

func main() {
	var number float32 = 0.12345
	fmt.Printf("%f \n", number)
}

// 0.123450
```



## float64

float64 最大能表示的值约为 1.6e308，可以使用 math.MaxFloat64 查看该值。

定义 float64 代码示例：

```
package main

import "fmt"

func main() {
	var number float64 = 0.12345
	fmt.Printf("%f \n", number)
}

// 0.123450
```





## 精度丢失

基本上所有编程语言的 float 类型都会产生精度丢失问题，这与计算机内部存储 float 的原理有关。

代码示例：

```
package main

import "fmt"

func main() {
	const PI = 3.141592653589793238
	fmt.Printf("%.18f \n", PI)
}

// 3.141592653589793116
```

解决方案是使用 decimal 类型来存储 float，但是 Golang 本身并未提供 decimal 类型，可以下载第三方包 decimal 获得此类型。



# 复数

## complex

复数在 Golang 中使用较少，其可分为 complex64 和 complex128，差别如下：

- complex64：实部和虚部为 32 位
- complex128：实部和虚部为 64 位

代码示例：

```
package main

import "fmt"

func main() {
	var number1 complex64 = 1 + 2i
	var number2 complex128 = 1 + 3i
	fmt.Printf("number1 : %#v \n", number1)
	fmt.Printf("number2 : %#v \n", number2)
}

// number1 : (1+2i)
// number2 : (1+3i)
```



# 科学计数

## 简单介绍

科学计数法最早由阿基米德提出，一个数可以表示成 a 与 10 的 n 次幂相乘的形式（1 ≤ | a | < 10，a 不为分数形式，n 为整数）。

在计算机领域里会用 e 来表示 10 的幂，下表是科学计数法在数学领域和计算机领域中的不同表现形式：

| 常规数        | 数学表示                  | 计算机表示 |
| ------------- | ------------------------- | ---------- |
| 700           | 7 x 10 <sup>2</sup>       | 7e+2       |
| 4,900,000,000 | 4.9 x 10 <sup>9</sup>     | 4.9e+9     |
| 3.1415        | 3.1415 x 10 <sup>0</sup>  | 3.1415e+0  |
| 0.00012       | 12 x 10 <sup>-5</sup>     | 12e-5      |
| -3.1415       | -3.1415 x 10 <sup>0</sup> | -3.1415e+0 |



## 代码演示

对于极大或极小数，请采取科学计数法来进行记录。这样的记录方式不论是从代码观感还是程序运行角度来说都十分不错。

如要记录全世界人口的总和（ 61 亿），我们可以使用 6.1e+9 进行存储。

代码示例：

```
package main

import "fmt"

func main() {
	var number int = 6.1e+9
	fmt.Printf("%d \n", number)
}

// 6100000000
```
